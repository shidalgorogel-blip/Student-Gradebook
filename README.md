# Student-Gradebook
Purpose of this program is to check, add, and view grades of student. This program is meant to be a student gradebook and access the grades of students.
When first prompted to the main page, you will be asked these options:
1. Add Student - Ability to add a student
2. Add Grade - Adds grade for said student
3. View Averages - Views averages for said student
4. View All Grades - Views all grades for said student
5. Exit -
Select an option (1-5):
When choosing, you will be asked the name of the student.
Will also be asked depending on the question what grade value you would like inputed.
